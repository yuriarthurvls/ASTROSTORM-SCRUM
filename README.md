# Sistema de Ocorrências Acadêmicas - IFPE

## Descrição

Este projeto foi desenvolvido como atividade da disciplina de Projeto e Prática utilizando Node.js, Express, Sequelize e MySQL.

O sistema tem como objetivo facilitar o cadastro e o gerenciamento de ocorrências acadêmicas de estudantes, permitindo o registro, consulta, edição e exclusão das informações cadastradas. Além disso o sistema possui um cadastro de usuários responsáveis pelo gerenciamento das ocorrências

---

## Funcionalidades

### CRUD de Ocorrências

- Cadastrar ocorrência
- Listar ocorrências
- Visualizar ocorrência
- Editar ocorrência
- Excluir ocorrência
- Pesquisar ocorrências por aluno
- Filtrar ocorrências por gravidade

### CRUD de Usuários

- Cadastrar usuário
- Listar usuários
- Visualizar usuário
- Editar usuário
- Excluir usuário
- Pesquisar usuário

## Como executar o projeto

1. Clone o repositório.

```
git clone https://github.com/yuriarthurvls/ASTROSTORM-SCRUM.git
```

2. Acesse a pasta do projeto.

```
cd ASTROSTORM-SCRUM
```

3. Instale as dependências.

```
npm install
```

4. Configure o banco de dados MySQL conforme as informações do arquivo de conexão.

5. Execute o projeto.

```
node app.js
```

6. Acesse no navegador.

```
http://localhost:3000
```
